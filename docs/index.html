Why OpenDicomViewer

Built different.

Native performance, clinical tools, and an architecture designed for customization.

Native Performance

Built with SwiftUI and Metal. No Electron, no web views — just native macOS speed with GPU-accelerated rendering.

Clinical Tools

Ruler, angle, and ROI measurements with real-time preview lines. Window/level controls with histogram display.

Multi-Panel Layouts

Side-by-side, stacked, and quad views with synchronized scrolling, zoom, and cross-reference lines.

MPR & Volume Rendering

One-click multiplanar reconstruction. GPU-powered MIP, MinIP, and average projections with adjustable slab thickness.

Open Source

MIT licensed. Fork it, customize it, use it however you want. No restrictions, no strings attached.

AI-Friendly Architecture

Clean, readable code with minimal abstraction. Designed to be extended with AI coding assistants like Claude or Copilot.

Features

Everything you need.
Nothing you don't.

Professional DICOM viewing capabilities in a lightweight, focused application.

Instant Image Display

Custom pure-Swift DICOM parser with incremental scanning. Images appear before the study finishes loading.

Multi-Panel Layouts

Single, side-by-side, stacked, and quad arrangements with drag-and-drop series assignment.

MPR Reconstruction

One-click multiplanar reformatting with axial, sagittal, coronal, and MIP views from any series.

GPU Volume Rendering

Metal compute shaders for real-time MIP, MinIP, and average intensity projections.

Window/Level Controls

Right-click drag, auto W/L, ROI-based W/L, and histogram display for precise contrast adjustment.

Measurement Tools

Ruler, angle, and ROI statistics with real-time dashed preview lines and millimeter-accurate measurements.

Cross-Reference Lines

See where other panels' slice planes intersect the current view for spatial orientation.

DICOM Tag Inspector

Browse all metadata tags for any image. Cursor readout shows HU values and patient coordinates in real-time.

Display Transforms

Invert, rotate, and flip images. Fit-to-window and reset-view shortcuts for rapid navigation.

JPEG 2000 Support

Handles compressed DICOM transfer syntaxes via DCMTK and OpenJPEG integration.

Keyboard-First

Fast by design.

Extensive keyboard shortcuts for every tool and action. Keep your hands on the keyboard.

Get Started

Up and running
in seconds.

1

Download

Grab the latest release, open the DMG, and drag OpenDicomViewer to your Applications folder. That's it.

Download DMG
2

Build from Source

Clone the repo and run the build script. Pre-built native libraries are included — no extra setup needed.

# Clone and build
git clone https://github.com/jnheo-md/open-dicom-viewer.git
cd OpenDicomViewer
./scripts/package_app.sh

# Install (optional)
cp -r OpenDicomViewer.app /Applications/
